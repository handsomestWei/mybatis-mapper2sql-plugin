# mybatis-mapper2sql-plugin
jetbrains intellij插件，自动提取mybatis mapper xml sql

## 简介
+ 自动识别mybatis mapper xml和提取sql，无需依赖dao class，轻松解决屎山项目。
+ 支持各类sql语法。
+ 自动mock sql参数。
+ 自动测试sql并记录执行结果。
